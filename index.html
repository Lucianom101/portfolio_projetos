Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimento web como o Streamlit e GitHub, foram utilizados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de restaurantes.
